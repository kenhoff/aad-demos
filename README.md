# aad-demos

## Running locally

1. Install node + git
2. fork
3. git clone
4. cd, npm install
5. node server.js
6. localhost:5000

## Contributing

1. Fork
2. Either:
  - make changes in your browser, or
  - clone your fork to your machine, make changes on your machine, and push to your fork
3. Open a Pull Request
